# Dayflow

AI-powered daily planning assistant built with Mastra.

## Quick Start

```bash
# Install dependencies
npm install

# Set up environment
cp .env.example .env
# Edit .env and add your OPENAI_API_KEY

# Run planning assistant
npm run dayflow
```

## Development

```bash
# Start Mastra Studio (http://localhost:4111)
npm run dev

# Type check
npm run typecheck

# Build
npm run build
```

## Features

- 🌅 Morning planning conversation
- 🤖 AI-powered task prioritization
- 📋 Implementation Intentions support
- 💾 Memory system (coming soon)

## Tech Stack

- [Mastra](https://mastra.ai) - AI Agent Framework
- TypeScript
- OpenAI GPT-4o-mini
